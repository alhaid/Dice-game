Welcome to the Pig Game
This is a game that has been designed by Hamzah Shafeeq a student at Sheridan college as an assignment for web programming class. It is a two player game, where the first player roll the dice and then have the option to either hold the number or take the risk and roll the dice again in order to get a higher score. In case you need more help about how to play the game please use the help button that is placed at the bottom right side of the game. Thank you for playing my game.
